# FeliStore Blockchain

Smart contracts for the FeliStore Web3 marketplace.

## Features

- Felicoin ERC-20 token for loyalty rewards
- Built with Hardhat
- OpenZeppelin contracts integration

## Getting Started

Install dependencies:
```bash
npm install
```

Compile contracts:
```bash
npm run compile
```

Run tests:
```bash
npm test
```

Deploy locally:
```bash
npm run node
# In another terminal:
npm run deploy
```
